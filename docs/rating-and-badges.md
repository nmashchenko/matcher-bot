# Rating & Badges

The rating system encourages healthy behavior and filters people properly. Two levels: quality signals (indirect indicators) and behavioral badges (how the person acts inside the product).

## Quality Signals (Metrics Tracked)

| Metric | Description |
|--------|-------------|
| Like Received Rate | Likes received per 100 impressions |
| Like Given Rate | Likes given per 100 impressions |
| Mutual Match Rate | Mutual matches from given likes |
| Chat Start Rate | % of matches where user actually enters/replies |
| Response Rate | Whether they reply during the 48-hour chat |
| Selectivity Balance | Balance between receiving and giving likes |
| Consistency | Regularity of activity |

## Behavioral Badges

### Attention Balance

| Badge | Criteria |
|-------|----------|
| **Attention Hunter** | Gets many likes, rarely likes back + low chat start/response rate |
| **Selective but Honest** | Gives few likes but has high mutual match rate and decent response rate |
| **Like Machine** | Likes too many profiles in a row, low mutual match rate |

### Communication

| Badge | Criteria |
|-------|----------|
| **Ghost** | Gets matches but often stays silent or doesn't start chats |
| **Solid Communicator** | Replies consistently, chats aren't empty |
| **Starter** | Frequently initiates and replies first |

### Profile Quality

| Badge | Criteria |
|-------|----------|
| **Empty Profile** | Minimal info — motivates filling it in (shown less otherwise) |
| **Clear Profile** | Bio + interests filled, normal activity, good engagement |

## How Badges Affect the System

- Badges are visible on profile cards so others can judge behavior upfront.
- Negative badges (Ghost, Attention Hunter, Like Machine) reduce visibility in feeds.
- Positive badges (Selective but Honest, Solid Communicator, Clear Profile) boost ranking.
